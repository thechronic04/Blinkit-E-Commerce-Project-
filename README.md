# Blinkit-E-Commerce-Project
This project analyses Blinkit’s Grocery Sales Data using Power BI.
The dashboard provides insights into sales performance, outlet performance, item categories, and customer ratings, helping businesses make better data-driven decisions.


# Project Objective
 To build an interactive Power BI dashboard that answers key business questions:

 Which item types generate the highest sales?
 How do outlet types and outlet sizes impact revenue?
 What is the trend of sales over the years?
 What is the contribution of fat content categories to total sales?
 Which outlet locations (Tier 1/2/3) perform best?

# Dataset Overview

# Insights Generated

Tier 3 locations generate the highest revenue.
High-size outlets contribute the most ($507K).
Regular fat items outperform low-fat items in sales.
Fruits & Vegetables and Snacks are top-performing categories.
Sales peaked in 2018 with a value of $205K.
